# 贡献者行为准则 / Code of Conduct

本项目采用 [Contributor Covenant](https://www.contributor-covenant.org/zh-cn/version/2/1/code_of_conduct/) v2.1 作为行为准则。

## 我们的承诺

为营造开放、友好的社区,我们承诺让每个人参与本项目和社区的体验都不受骚扰,无论其年龄、体型、可见或不可见的残疾、种族、性别、经验水平、教育程度、国籍、外貌、宗教或性取向。

## 行为标准

有助于营造积极环境的行为包括:表达善意与同理、尊重不同观点、礼貌地给予和接受反馈、为错误承担责任并致歉、关注社区整体利益。

不可接受的行为包括:性化的言语或图像、挑衅/侮辱性评论与人身攻击、公开或私下骚扰、未经许可公布他人隐私信息,以及其他在专业场合中合理认定为不当的行为。

## 执行

维护者有权移除、编辑或拒绝不符合本准则的评论、提交、代码、issue 等贡献。如遇违规行为,可通过 GitHub 私信仓库所有者举报,所有投诉都将被审查并妥善处理。

---

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1. Report unacceptable behavior to the repository owner via GitHub.
